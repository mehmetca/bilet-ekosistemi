# Salon planı görselleri

Bu klasöre salon planı görselini (fotoğraf veya PDF’den export edilmiş PNG/JPEG) koyun.

## Theater Duisburg (veya benzeri plan)

- Dosya adı: **theaterduisburg.png**
- Görseli bu isimle `public/seatplans/` altına kopyalayın.
- Etkinlikte kullandığınız oturum planının adı "Theater Duisburg" (veya "theater duisburg") içeriyorsa, bu görsel üzerinde tıklanabilir koltuklar gösterilir.

Görsel yoksa sayfa yine çalışır; liste görünümü veya genel salon planı kullanılır.
